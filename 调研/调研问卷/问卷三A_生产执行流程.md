**综合判定优先级：**

**说明：** 当多个检测项（针检、表检、板型）的结果不一致时，如何确定料卷的最终质量等级？

**常见的判定方式有两种：**

#### 方式一：最严格指标优先（木桶原则）

**逻辑：** 哪个指标最差，整体就是什么等级。就像木桶的短板决定了能装多少水。

| 针检等级 | 表检等级 | 板型等级   | 最终等级   | 判定说明             |
| -------- | -------- | ---------- | ---------- | -------------------- |
| A级      | A级      | A级        | **A级**    | 全部最优，结果最优   |
| A级      | **B级**  | A级        | **B级**    | 以最差的表检为准     |
| A级      | B级      | **C级**    | **C级**    | 以最差的板型为准     |
| A级      | A级      | **不合格** | **不合格** | 有一项不合格即不合格 |

#### 方式二：加权评分综合判定

**什么是加权评分？**
加权评分是指给不同的检测指标设定**不同的重要程度（权重）**，然后**综合计算**得出最终分数和等级。权重越高，该指标对最终结果的影响越大。

**计算方法：**

1. 先根据重要性给每个指标分配权重（所有权重相加=100%）
2. 给每个等级赋分（如：A级=100分，B级=80分，C级=60分，不合格=0分）
3. 综合得分 = 各指标得分 × 权重 之和
4. 根据综合得分判定最终等级

**详细示例：**

假设权重设定为：针检40% + 表检30% + 板型30% = 100%

| 检测项   | 检测结果     | 等级得分 | 权重 | 加权得分       |
| -------- | ------------ | -------- | ---- | -------------- |
| 针检     | 6个/㎡ → B级 | 80分     | 40%  | 80×40% = 32分  |
| 表检     | 无缺陷 → A级 | 100分    | 30%  | 100×30% = 30分 |
| 板型     | 3mm → A级    | 100分    | 30%  | 100×30% = 30分 |
| **合计** | -            | -        | 100% | **92分**       |

**综合得分判定规则示例：**

- 90-100分 → A级
- 80-89分 → B级
- 60-79分 → C级（待评审）
- <60分 → 不合格

上例中综合得分92分 → 判定为**A级**

### 3.2 订单与料卷绑定关系

**说明：** 订单与料卷绑定是指确定用哪些料卷来生产哪个订单，建立对应关系。这是生产管理的关键环节，直接影响物料追溯、成本核算和交期管理。

**⚠️ 重要概念说明：本节中的“料卷”指的是什么？**

在铝箔生产过程中，物料会经历不同阶段，每个阶段的“卷”有不同的名称：

| 名称               | 定义                                       | 来源       | 去向                 |
| ------------------ | ------------------------------------------ | ---------- | -------------------- |
| **母卷**           | 轧机轧制出的原始卷材，宽度较大             | 轧机轧制   | 送往分切机           |
| **料卷（分切卷）** | 母卷经过分切机切割后的较窄卷材，已完成质检 | 分切机切割 | 送往精切机或直接包装 |
| **成品卷**         | 精切后的最终产品，符合客户规格要求         | 精切机切割 | 包装入库、发货       |

```
生产流程：轧机母卷 → 分切机切割 → “料卷”（分切卷） → 精切机切割 → 成品卷
                                      ↑
                            本节所说的“料卷”指的就是这个！
```

**本节中的“料卷”均指「分切后的料卷」**，即已经完成分切、质检、评审，等待精切的半成品卷材。排产员根据订单要求，从这些料卷中挑选合适的来安排精切生产。

---

**举例说明：**

- 客户订单CO20250108001要求生产1000米宽度1200mm的铝箔
- 排产员在系统中找到2个符合要求的**分切料卷**：FC20250105001（600米）和FC20250105002（500米）
- 将这2个料卷绑定到订单CO20250108001，建立对应关系
- 后续精切生产时，操作工扫描料卷号，系统自动显示对应的订单信息和加工要求

### 3.7 排产批量和换型规则

**说明：** 为了提高生产效率，排产时需要考虑批量和换型。

**什么是批量？** 批量是指连续生产同一种规格产品的数量。批量越大，换型次数越少，效率越高。

**什么是换型？** 换型是指从生产一种规格切换到另一种规格时，需要调整设备（如调刀）。换型需要时间，这段时间设备不能生产。

**举例：**

- 订单A：宽度1200mm，5卷
- 订单B：宽度1000mm，3卷
- 订单C：宽度1200mm，8卷
- 好的排产：A+C→B（相同规格合并，只换型1次）
- 不好的排产：A→B→C（换型2次，浪费时间）

#### 3.7.1 批量规则

**说明：** 批量规则用于控制生产批次的大小，避免频繁换型或批量过大。

| **小于最小批量如何处理** | 【填写】（如：合并其他订单凑够批量、延后排产等待更多订单、特殊情况下允许小批量生产等） |

**换型顺序说明：**

"换型顺序"是指：当需要连续生产不同规格的产品时，按什么顺序切换更省时省力？

**示例：**

- **宽度切换**：假设要生产1200mm、1000mm、800mm三种宽度
  - 从宽到窄（1200→1000→800）：只需要逐步缩小刀距，操作简单
  - 从窄到宽（800→1000→1200）：每次都要扩大刀距并重新校准，耗时更长
  - **结论**："从宽到窄"更高效
- **厚度切换**：假设要生产0.02mm、0.015mm、0.01mm三种厚度
  - 从厚到薄：张力调整相对容易
  - 从薄到厚：需要更大调整，可能需要换辊
  - **结论**："从厚到薄"更高效

**贵公司的换型顺序要求：**
