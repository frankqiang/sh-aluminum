# 调研问卷使用顺序

## 第一天

| 时间 | 活动     | 使用问卷           | 对象              |
| ---- | -------- | ------------------ | ----------------- |
| 上午 | 车间参观 | 问卷一（边看边问） | 生产/设备人员陪同 |
| 下午 | 流程访谈 | 问卷三A + 问卷三B  | 生产部 + 质量部   |

## 第二天

| 时间 | 活动         | 使用问卷       | 对象       |
| ---- | ------------ | -------------- | ---------- |
| 上午 | 系统对接访谈 | 问卷二         | IT部       |
| 上午 | 设备细节补充 | 问卷一（补充） | 设备部     |
| 下午 | 需求确认     | -              | 双方负责人 |

## 回来后

将问卷发给客户，让各部门补充您没问全的细节（3-5个工作日）。
