## 二、设计方案

### 2.1 订单匹配推荐 — 二级弹窗卡片列表

#### 交互流程

当前的下拉菜单信息承载力不足，改为**"选择订单"按钮 + 二级弹窗**模式：

```
┌─── 新增分切计划（主弹窗）─────────────────────────┐
│ ① 来料选择    [Y2601475 ▼]   [1#分切机 ▼]        │
│ ② 质量评审    正常分切 · 一级品 · 有效宽度1596mm  │
│ ③ 切刀方案设计                                     │
│   边丝  │  —    │  10mm │ ...                      │
│   订单  │ [📋 选择订单]  │ ...      ← 点击此按钮   │
│   边丝  │  —    │  10mm │ ...                      │
│ ④ 计划说明                                         │
└───────────────────────────────────────────────────┘

                    点击后弹出 ↓

┌─── 订单匹配推荐（二级弹窗）──────────────────────┐
│ 筛选条件：合金 1060 · 宽度 ≤1596mm · 厚度 13μm  │
│ 排序：[匹配度推荐 ▼]                              │
│                                                    │
│ ┌── 推荐 ★★★★★ ──────────────────────────────┐  │
│ │ 🔴 SZ25121602434 · 标兵母卷                  │  │
│ │ 宽度 1585mm · 厚度 12μm                      │  │
│ │ 📅 交期 3.08 (剩3天)  欠单 5卷               │  │
│ │ 管芯 152.4/壁厚4                    [选择]   │  │
│ └──────────────────────────────────────────────┘  │
│ ┌── 推荐 ★★★★☆ ──────────────────────────────┐  │
│ │    SZ25121602435 · 标兵母卷                   │  │
│ │ 宽度 1340mm · 厚度 13μm                      │  │
│ │ 📅 交期 3.10 (剩5天)  欠单 6卷               │  │
│ │ 管芯 152.4/壁厚4                    [选择]   │  │
│ └──────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────┘
```

选中后自动关闭二级弹窗，订单信息回填到主弹窗的订单段中。

#### 推荐排序算法

将 `findBestOrder()` 从"宽度最优"升级为"多维度加权推荐"：

```
1. 过滤条件：
   - 合金匹配（必须一致）
   - 宽度 ≤ 有效宽度
   - 厚度匹配（新增）
   - 质量等级匹配（新增，基于 acceptGrade 字段）

2. 排序权重：
   - 交期紧迫度（剩余天数越少权重越高）  → 40%
   - 急单标记（priority === 'urgent'）    → 20%
   - 宽度贴合度（越接近有效宽度越好）     → 25%
   - 欠单比例（remainingCoils/totalCoils） → 15%
```

#### 卡片信息结构

每张订单卡片展示以下字段：

| 区域     | 内容                       |
| -------- | -------------------------- |
| 头部标识 | 急单红点 + 订单号 + 客户名 |
| 规格信息 | 宽度 × 厚度，管芯规格      |
| 紧迫度   | 📅 交期日期 + (剩N天)      |
| 进度     | 欠单卷数：已切/总需        |
| 匹配度   | ★ 评分（基于加权排序分数） |
| 操作     | [选择] 按钮                |

#### 数据变更

订单池 `order-pool.js` 需增加字段：

```js
{
  acceptGrade: ['一级品', '二级品'],  // 可接受的质量等级
  // ...其他已有字段
}
```

### 2.2 切刀方案计算 — 横截面可视化

#### 切刀横截面示意图

在段列表下方增加**可视化条形图**，实时反映段分配情况，同时叠加质量缺陷标注：

```
母卷横截面 1210mm
                                    ↓ 气道@670mm（避留白）
├── 边丝 ──┤───── 订单A ──────┤ ⚡ ┤──── 订单B ─────┤── 边丝 ──┤
    10mm         660mm                  516mm           10mm
    灰色          蓝色         黄色标记    蓝色            灰色
```

**颜色编码**：

- 灰色 = 边丝
- 蓝色 = 订单段
- 红色 = 切除段（不可用区域）
- 橙色 = 余料段
- 黄色标记/闪烁 = 避留白位置（可用但切刀不能落在此处）

**交互**：

- 每个段按实际宽度**等比缩放**
- 鼠标悬浮显示段详情（订单号、客户、宽度、备注）
- 段宽度变更时条形图实时更新

#### 余料段自动生成

当所有边丝 + 订单 + 切除段分配完后，剩余宽度自动创建一个"余料"段：

- 段类型 = `waste`
- 标注文字 = "可配其他订单"或"废料"
- 余料宽度 > 某阈值时（如 >100mm）高亮提示"可尝试搭配窄幅订单"

#### 边丝范围校验

- 默认 10mm
- 合理范围：5-20mm
- 超出范围时橙色警告提示："边丝宽度建议 5-20mm"

#### 避留白冲突校验

切刀方案设计时需校验：

- 遍历评审指令中 `action === '避留白'` 的位置
- 检查所有订单段的边界是否与避留白位置重叠（±5mm 容差）
- 冲突时橙色警告："订单段边界与避留白位置冲突，请调整"

### 2.3 质量匹配建议 — 等级联动 + 缺陷可视化

#### 质量等级→订单匹配联动

**核心逻辑**：母卷评审等级决定可匹配的订单范围。

```
母卷等级 = 一级品 → 显示全部订单
母卷等级 = 二级品 → 仅显示 acceptGrade 包含 '二级品' 的订单
```

**UI 表现**：

- 二级弹窗筛选条件区域自动增加"质量等级：二级品"标签
- 不符合等级的订单卡片置灰并标注"⚠️ 此订单仅接受一级品"
- 质量评审区域增加醒目提示：

```
⚠️ 此卷已降为二级品，可匹配的订单范围已自动收窄
   二级品可配订单：3个 | 已自动排除一级品专属订单
```

#### 缺陷位置可视化

缺陷位置标注**与切刀横截面示意图合并**，在同一条形图上展示：

```
母卷横截面 1210mm

缺陷层：
  Q侧 840-990mm 针孔(切除)        C侧 0-30mm 针孔(切除)
  ├───── 红色填充 ─────┤           ├─ 红色 ─┤

切刀层：
├ 边丝 ┤──── 订单A ────┤── 切除 ──┤── 切除 ─┤─ 边丝 ┤
  10mm      1030mm        150mm      30mm      10mm
```

这样计划员可以直观看到：

- 缺陷在母卷的什么位置
- 切除段是否对应了缺陷位置
- 订单段是否落在了"干净"的区域

#### 降级品流转提示

评审等级为二级品时，在 `② 质量评审` 区域增加：

- 背景色变为浅黄色
- 增加 `⚠️` 图标和说明文字
- 订单推荐弹窗中自动过滤

### 2.4 UI/UX 改进

#### "以单找料"反向操作入口

在分切排产主页面增加**双视角切换**：

```
┌─────────────────────────────────────────────┐
│  [📦 以料排产]   [📋 以单找料]              │
│       ↑ 当前视角（默认）                     │
└─────────────────────────────────────────────┘
```

- **以料排产**（当前已有）：展示母卷列表，为每卷安排订单
- **以单找料**（新增）：展示待交付订单列表，点击某订单后反向推荐可匹配的母卷

"以单找料"视角的订单列表应包含：

- 订单号、客户、规格、交期、剩余卷数
- 按交期紧迫度排序
- 点击后展示可匹配的母卷卡片列表

#### 主列表总览指标

在现有统计条中增加：

```
过滤结果: 14  |  已完成: 5  |  进行中: 3  |  待评审: 2  |  已排定: 4
                                                     ↓ 新增
                              |  完成率: 35.7%  |  平均废料率: 8.2%
```

支持按机台分组/折叠展示：

```
▼ 1#分切机 (6条)
  SP001  Y2601301  1060  ...
  SP002  Y2601302  1100  ...
▼ 2#分切机 (3条)
  SP007  Y2601315  8079  ...
```
