# 排产模块原型设计文档

> **日期**：2026-03-05
> **目标**：为客户演示的高保真可交互原型
> **技术栈**：Vite + Vue
> **设计风格**：浅色主题，经典ERP表格风格
> **数据**：使用调研中的真实卷号、客户名、合金型号

---

## 一、设计决策汇总

| 决策项   | 结论                                        |
| -------- | ------------------------------------------- |
| 原型目标 | 说服客户，高保真可交互                      |
| 技术形式 | Vite + Vue 项目                             |
| 首页     | 生产计划总览 Dashboard                      |
| 页面范围 | Dashboard + 轧机 + 分切 + 精切（共4个页面） |
| 设计风格 | 浅色主题，干净专业                          |
| 切刀方案 | 表格展示（不做图形化）                      |
| 模拟数据 | 使用真实数据（卷号、客户名等）              |
| 整体方案 | 方案A：经典ERP表格风格                      |

---

## 二、全局框架

### 2.1 顶部栏

```
┌──────────────────────────────────────────────────────────────────┐
│  🏭 神火新材 · 智能生产管理系统          🔔 通知(3)  👤 张计划  │
└──────────────────────────────────────────────────────────────────┘
```

- 左侧：企业logo + 系统名称
- 右侧：通知图标（预警消息数）+ 用户头像/姓名
- 高度：约56px，浅色背景，底部细线分隔

### 2.2 左侧边栏

```
┌────────────────┐
│                │
│ 📋 排产管理  ▼ │  ← 一级菜单（当前模块，展开状态）
│   📊 总览     │  ← 二级菜单
│   🔧 轧机排产 │
│   ✂️ 分切排产 │  ← 当前页面高亮
│   🔬 精切排产 │
│                │
│ 🔍 质量管理  ▸ │  ← 一级菜单（收起状态，置灰）
│                │
│ 📦 物料管理  ▸ │  ← 一级菜单（收起状态，置灰）
│                │
│ ⚙️ 生产执行  ▸ │  ← 一级菜单（收起状态，置灰）
│                │
│  ─────────────│
│  ◀ 收起侧边栏 │  ← 可收起为图标模式
└────────────────┘
```

- 宽度：展开220px，收起64px（仅图标）
- 一级菜单：排产管理、质量管理、物料管理、生产执行
- 二级菜单（排产下）：总览、轧机排产、分切排产、精切排产
- 当前页高亮：左侧色条 + 背景色
- 不可用模块：质量/物料/执行置灰，hover提示"即将上线"
- 底部收起按钮

### 2.3 整体组合

```
┌──────────────────────────────────────────────────────────────────┐
│  🏭 神火新材 · 智能生产管理系统          🔔 通知(3)  👤 张计划  │

│                │                                                 │
│ 🔍 质量管理  ▸ │                                                 │
│ 📦 物料管理  ▸ │                                                 │

```

## 五、设计原则

| 原则         | 说明                                                        |
| ------------ | ----------------------------------------------------------- |
| 浅色主题     | 白色背景+灰色边框+蓝色主色调                                |
| 表格为核心   | 数据密度优先，接近Excel使用习惯                             |
| 状态色彩统一 | 🟢绿=完成/运行, 🔵蓝=进行中, 🟡黄=待料/警告, 🔴红=停机/逾期 |
| 右侧详情面板 | 列表行点击后右侧滑出详情，无需跳转页面                      |
| 修改留痕     | 每条计划都有修改记录区域                                    |
| 评审联动     | 分切列表中直接展示评审状态，详情中展示完整评审信息          |
