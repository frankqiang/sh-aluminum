# 轧机排产 - 新增计划功能设计

> **日期**：2026-03-05
> **范围**：轧机排产页面的"新增计划"弹窗交互
> **目标**：让客户演示时看到完整的"新增排产计划"交互流程

---

## 一、设计决策

| 决策项       | 结论                                        |
| ------------ | ------------------------------------------- |
| 新增粒度     | 一次新增 = 一卷完整轧制计划                 |
| 交互形式     | 弹窗表单（Modal Dialog）                    |
| 表单结构     | 分步式（Step 1 基础信息 → Step 2 道次配置） |
| 道次填写方式 | 手动填写（非自动推算）                      |
| 演示深度     | 仅展示交互流程，提交后 Toast 提示成功       |

---

## 二、交互流程

```
[+ 新增计划] 按钮
      ↓
  弹窗打开 → Step 1: 基础信息
      ↓ 点击 [下一步]
  弹窗切换 → Step 2: 道次配置 + 工艺说明
      ↓ 点击 [确认提交]
  弹窗关闭 → Toast 提示"计划添加成功"
```

## 四、Step 2：道次配置 + 工艺说明

```
┌──────────────── 新增轧机计划 ──── (2/2) ── ✕ ─┐
│                                                  │
│  ── 道次配置 ──────────────────────────────────  │
│                                                  │
│  当前机台: 1号粗轧机（可轧 1-3 道次）            │
│                                                  │
│  道次 │ 目标厚度(μm) │         │                 │
│  ─────┼──────────────┼─────────│                 │
│  P1   │ [  98  ]     │  [🗑]  │                 │
│  P2   │ [  40  ]     │  [🗑]  │                 │
│  P3   │ [  16  ]     │  [🗑]  │                 │
│                                                  │
│  [+ 添加道次]                                    │
│                                                  │
│  ── 工艺说明 ──────────────────────────────────  │
│  ┌──────────────────────────────────────────┐    │
│  │ 针孔≤1000 厚度不超6                      │    │
│  │ 注意板型及表面质量                        │    │
│  │                                          │    │
│  └──────────────────────────────────────────┘    │
│                                                  │
│              [← 上一步]    [确认提交]            │
└──────────────────────────────────────────────────┘
```

### 交互说明

| 交互          | 说明                                                 |
| ------------- | ---------------------------------------------------- |
| 当前机台提示  | 根据 Step 1 选的机台，显示该机台可轧的道次范围作参考 |
| [+ 添加道次]  | 点击新增一行，道次号自动递增（P1→P2→P3...）          |
| [🗑] 删除按钮 | 删除该道次行                                         |
| 工艺说明      | 多行文本框，自由输入，选填                           |
| [← 上一步]    | 返回 Step 1，已填数据保留                            |
| [确认提交]    | 关闭弹窗 + 弹出成功 Toast                            |

## 五、组件结构

```
AddPlanModal.vue          ← 弹窗主组件
├── Step 1: 基础信息表单
├── Step 2: 道次配置 + 工艺说明
├── 步骤指示器 (1/2, 2/2)
└── Toast 提示组件
```

## 六、设计原则

- **原型定位**：仅展示交互流程，提交后不真实写入数据
- **风格统一**：弹窗使用现有设计系统的变量（颜色、圆角、阴影）
- **步骤之间数据保留**：切换步骤不丢失已填内容
- **机台感知**：Step 2 根据所选机台显示可轧道次提示
