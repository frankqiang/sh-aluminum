# MES系统交互原型 实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 构建一套覆盖一期/二期/三期所有功能的MES系统交互原型，用于客户演示和方案展示。

**Architecture:** 纯HTML/CSS/JS的多页面应用，共享设计系统和模拟数据层。各页面通过侧边栏导航切换，使用模拟数据展示完整交互流程。工业深色主题风格，深蓝/暗灰为底色，状态色（绿/黄/红/蓝）点缀。

**Tech Stack:** HTML5 + Vanilla CSS + Vanilla JavaScript，无框架依赖。Google Fonts（中文字体）。

## 设计规范

### 色彩系统

```css
/* 基础色 */
--bg-primary: #0f1923; /* 主背景 - 深蓝黑 */
--bg-secondary: #1a2636; /* 卡片背景 */
--bg-tertiary: #243447; /* 输入框/悬停 */
--border: #2d4a5e; /* 边框 */

/* 文字色 */
--text-primary: #e8edf2; /* 主文字 */
--text-secondary: #8899aa; /* 次要文字 */
--text-muted: #556677; /* 禁用/提示 */

/* 状态色 */
--color-success: #00d68f; /* 合格/完成 */
--color-warning: #ffaa00; /* 预警/待处理 */
--color-danger: #ff3d71; /* 不合格/紧急 */
--color-info: #0095ff; /* 信息/进行中 */
--color-accent: #7b61ff; /* 强调/焦点 */

/* 分期标记色 */
--phase-1: #0095ff; /* 一期 - 蓝 */
--phase-2: #00d68f; /* 二期 - 绿 */
--phase-3: #7b61ff; /* 三期 - 紫 */
```

### 排版

- 中文字体：`"Noto Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif`
- 数字字体：`"JetBrains Mono", "Fira Code", monospace`
- 标题: 20px/600, 正文: 14px/400, 小字: 12px/400
- 车间终端: 标题 32px, 正文 24px, 按钮 80×60px

### 组件规范

- 卡片: `border-radius: 8px; border: 1px solid var(--border);`
- 按钮: 高36px, 圆角6px; 车间终端按钮高60px, 圆角12px
- 表格: 行高44px, 斑马纹(奇偶行微妙色差)
- 标签: 圆角12px, 内边距4px 10px
- 输入框: 高36px, 背景var(--bg-tertiary)

- 订单池（15条，含不同状态、客户、规格）
